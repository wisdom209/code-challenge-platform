Turn-in directory: c04/ex05/

Files to turn in: ft_atoi_base.c

Allowed functions: none specified

Prototype: int ft_atoi_base(char *str, char *base);

Description: Write a function that converts the initial portion of the string pointed by str to int representation, where str is in a specific base given as a second parameter. Except for the base rule, the function should work exactly like ft_atoi. If the base is invalid (empty, size 1, duplicate characters, or contains '+', '-', or whitespace), the function should return 0.
