Turn-in directory: c04/ex03/

Files to turn in: ft_atoi.c

Allowed functions: none specified

Prototype: int ft_atoi(char *str);

Description: Write a function that converts the initial portion of the string pointed by str to its int representation. The string can start with an arbitrary amount of white space, followed by an arbitrary amount of + and - signs, and then digits. The function should read until the string stops following the rules and return the number found until now. Do not handle overflow or underflow.
