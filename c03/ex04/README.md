Turn-in directory: c03/ex04/

Files to turn in: ft_strstr.c

Allowed functions: none specified

Prototype: char *ft_strstr(char *str, char *to_find);

Description: Reproduce the behavior of strstr (man strstr). Finds the first occurrence of the substring to_find in str. Returns a pointer to the beginning of the located substring, or NULL if not found. In these tests, the return value is given as the offset from the start of str (or -1 for NULL).
